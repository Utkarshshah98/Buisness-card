## 🚀 Getting Started

1. **Clone the repository**
   ```bash
   git clone https://github.com/your-username/business-card.git
   cd business-card
   ```

2. **Open in browser**
   ```bash
   # Just open the HTML file directly
   open buisnesscard.html
   ```
   Or drag and drop `buisnesscard.html` into your browser.

> ⚠️ Make sure `7.jpg` and `styles.css` are in the **same folder** as the HTML file.
